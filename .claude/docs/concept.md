# Paradise 2033 — Концепт

> Источник правды по идее, сюжету и тону. Если что-то конфликтует между файлами — этот прав.
> История портирована из Unity-проекта `D:\UnityProjects\Paradise2033` (см. там `.claude/docs/concept.md` для расширенного каталога контента и fail-сценариев на полную игру).

## Жанр и референс

- **Жанр:** idle/tap-кликер с престижем.
- **Геймплейный референс:** корейский *Don't Get Fired!* — облачка-задания, тап тратит HP/энергию, шанс на повышение/провал, делегирование задач агентам.
- **Тематический референс:** сатира на эпоху AI-агентов — каждое действие в жизни должно выполняться через подключённого AI; кто не подключился — на помойке.

## Сеттинг

**2033 год, постнейросетевая антиутопия.** Глобально — работа, отношения, бытовуха, секс выполняются через AI-агентов. Принципиальные «солисты» — на свалке общества.

**Главный герой:** безымянный бомж в комнате-помойке с древним ноутбуком. Уволили, девушка ушла, собака уцелела. Принципиально отказывается от агентов — «сам справлюсь».

**Антагонист / quest-giver:** **«Голос Сверху»** — текстовый интерфейс в духе Telegram-бота, всплывающий внутри игры. Спокойно-ироничная манера, говорит «по-айтишному» с лёгким матерком. Продавец агентов и менеджер по продажам имплантов. По мере прогрессии — всё настойчивее.

## Тон

- Сухая ирония, не клоунада.
- Бытовой мат в репликах Голоса и фоновых надписях — уместен, но не в каждой строке.
- Сатира на: AI-стартапы, корпоративный язык (synergy, empower, AI-first), социальные сети, доставку, подписочные модели.
- **Не делаем:** мрак ради мрака, чернуху, прямую политическую сатиру.

## Нарративная структура MVP (три фазы)

### Run 1 — соло, без агентов

ГГ просыпается в помойке. Голос предлагает «ржавую Алису» бесплатно для первой сделки. Единственный диалог-выбор MVP: «Сам справлюсь». (Второй вариант возвращает к этому же экрану с ремаркой Голоса.)

**Геймплей (3–7 мин):** облачки `Срочный заказ` (комп) и `Гулять с собакой`. Тап тратит энергию. Цель — $400 для перехода в Run 2.

**Триггер 30–40%:** Голос возвращается с повторным предложением.

**Финал:**
- Провал (энергия = 0 или fail-roll) → fade-to-black, диалог «Знал. Держи Алису бесплатно».
- Успех (теоретически ~5–15%) → ремарка «Упрямый. Посмотрим как Run 2 потащишь».

### Run 2 — с одним агентом

В магазе уже «Алиса (ржавая)» — таскает заказ автоматически (медленно, без энергии). Собаку гулять надо самому.

**Fail-сценарий MVP:** *«Алиса утекла промптом твою банковскую карту. Штраф $200»*. Это и есть сатира на дырявые агенты — игрок смеётся, потом понимает, что его всё равно вынудят покупать апгрейд.

**Триггер на провал:** Голос предлагает Tinder-агента ИЛИ имплант хирурга. Магаз открывает раздел «Импланты».

### Prestige 1 — финал MVP

В магазе разблокирован «Ржавая правая рука v0.1». Покупка → fade-to-black + «Хирург-агент завершил операцию. Болит. Но скорость работы +50%». **Без визуальной замены спрайта в MVP** — только UI-индикатор и числовой бафф.

Финальный экран: «To be continued in full release».
